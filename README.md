# MT5
Проект для торговли на MetaTrader 5 используя python



## Установка
1. Создать виртуальное окружение.
2. Установить зависимости из requirements.txt.
3. Заполнить config/settings.json и добавить credentials.
4. Проверить путь ярлыка MT5 AI Trader.lnk к файлу terminal64.exe (запуска самого МТ5)

Содержаание credentials.json (для подключения ме5)
```json
{
    "login": "",   //Логин
    "password": "",   //Пароль
    "server": ""   //Имя сервера
}
```



Обозначение настроек settings.json
```json

{
  "system": {
    "shutdown_command": "all off",   // команда для отключения системы, можно свою
    "connection_monitor_interval": 30,  // интервал проверки подключения (в секундах)
    "reconnection_delays": [0, 10, 30, 60, 300]  // задержки между попытками переподключения (в секундах)

  "display": {
    "terminal_info": true,    // показывать информацию о терминале MT5
    "account_info": true,     // показывать информацию о торговом счёте
    "all_symbols": true,     // получать все доступные символы из MT5
    "symbol_details": false, // показывать подробную информацию по конкретному символу
    "grouped_symbols": false //  группировать и отображать символы по категориям false true
  },

  "history": {      // Получение осторических данных
    "enabled": true,
    "start_date": "2026-06-08", // год, месяц, день
    "end_date": "2026-06-09", // на данный момен обязательное значение
    "timeframe": "M1",     //свечи 1 минутная свеча
    "symbols": ["GBPUSD", "EURUSD"],
    "mode": "custom",
    "symbols_directories": "off" //Выключено на данный момен, Создание директорий в history  "all" - всё; ["GBPUSD"]/
  }
}
```
